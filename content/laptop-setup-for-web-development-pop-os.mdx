## Contents

## Operating System: Pop!\_OS 22.04 LTS

> Download Pop!\_OS here: https://pop.system76.com

### Installation Checklist

1. Update and upgrade the system.

   ```sh
   sudo apt update
   sudo apt upgrade
   sudo apt dist-upgrade
   ```

2. Update the firmware (if available).

3. Install proprietary media codecs and fonts.

   These are the commonly used proprietary software such as codecs for playing media formats (MP3, AVC, etc.) and fonts (Arial, Times New Roman, etc.).

   ```sh
   sudo apt install ubuntu-restricted-extras
   ```

4. Install and enable the firewall.

   To install the GUI:

   ```sh
   sudo apt install gufw
   ```

5. Install TLP for better power management.

   TLP is a free, open-source, and feature-rich utility for [optimizing battery consumption on laptops running Ubuntu](https://vitux.com/improving-battery-life-in-ubuntu-with-tlp) and other Linux distributions.

   ```sh
   sudo apt install tlp
   sudo tlp start
   sudo tlp-stat -s # view detailed system information and status of the TLP utility
   ```

6. Enable TRIM for SSD drives.

   Enabling TRIM can help maintain the performance of SSD over time, extending its life.

   ```sh
   sudo systemctl enable fstrim.timer
   ```

### General Settings

1. Show battery percentage.

2. Set ALT+TAB behavior to switch windows, not applications.

   Go to Keyboard Shortcut Settings. Search for ‘Switch windows’ and map ALT+ TAB to it (replacing the existing mapping for switching apps).

3. Set natural scrolling in mouse and touchpad scroll.

4. Display weekday in date and time (Alternatively, use the GNOME tweaks tool GUI).

   ```sh
   gsettings set org.gnome.desktop.interface clock-show-weekday true
   ```

## Terminal / Shell

> Dotfiles: https://github.com/jromest/dotfiles

1. Update `git` and create a `.gitconfig` config file.

   Version control system that tracks changes in files.

   ```sh
   sudo apt install git
   nano .gitconfig
   ```

2. Install `zsh`.

   Zsh (Z Shell) is a powerful shell with advanced features and customization options, often used as an alternative to the more common Bash shell.

   ```sh
   sudo apt install zsh
   ```

   - Set `zsh` as the default shell (you need to log out and log in to take effect).

     ```sh
     chsh -s $(which zsh)
     ```

   - Install `zinit` (zsh plugin manager) and create a `.zshrc` config file.

     https://github.com/zdharma-continuum/zinit

     ```sh
     nano .zshrc
     ```

3. Install `tmux`.

   Tmux is a terminal multiplexer that allows multiple terminal sessions to be accessed simultaneously in a single window.

   ```sh
   sudo apt install tmux
   ```

   - Install `tpm` (TMUX plugin manager) and create a `.tmux.conf` config file.

     https://github.com/tmux-plugins/tpm

     ```sh
     nano .tmux.conf
     ```

4. Install `fzf`.

   Command-line fuzzy finder that allows for quick and efficient searching of files and directories.

   https://github.com/junegunn/fzf

5. Install `ripgrep`.

   Ripgrep (rg) is a line-oriented search tool that recursively searches the current directory for a regex pattern.

   https://github.com/BurntSushi/ripgrep

6. Install `bat`.

   A cat clone with syntax highlighting and git integration.

   https://github.com/sharkdp/bat

7. Install `diff-so-fancy`.

   Diff highlighting tool that makes diffs more human-readable and visually appealing.

   https://github.com/so-fancy/diff-so-fancy

8. Install `nnn`.

   A full-featured, tiny, nearly zero-config and incredibly fast terminal file manager.

   https://github.com/jarun/nnn

9. Install `tldr`.

   Simplified and community-driven man page that provides practical examples of command usage.

   https://tldr.sh/

## Development Setup

1. Install NodeJS via `nvm` or Node Version Manager.

   https://github.com/nvm-sh/nvm

2. Install `yarn` classic (via Debian package repository).

   https://classic.yarnpkg.com/lang/en/docs/install/#debian-stable

3. Install VS Code.

   https://code.visualstudio.com

   VS Code extensions:

   1. Night Owl (theme)
   2. ESLint
   3. Prettier
   4. Code Spell Checker
   5. Auto Rename Tag
   6. Pretty Typescript Errors
   7. Quick Lint JS
   8. Tailwind CSS IntelliSense
   9. Git Lens
   10. Docker
   11. GraphQL: Language Feature Support
   12. GraphQL: Syntax Highlighting (automatically downloaded by Language Feature Support)
   13. TODO Highlight
   14. Vim

4. Install Vim.

   ```sh
   sudo apt install vim
   ```

   - Install `vim-plug` (vim plugin manager).

     https://github.com/junegunn/vim-plug

   - Create a vim config and create a tmp directory.

     ```sh
     vim .vimrc
     mkdir .vim/tmp
     ```

5. Install Docker.

   https://docs.docker.com/engine/install/ubuntu

   **Note:** Reboot if having permission error to run docker without sudo.
